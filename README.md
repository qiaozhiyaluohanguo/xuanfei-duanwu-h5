# 宣肺端午祝福 H5 小游戏

## 打开方式

直接打开 `index.html` 即可预览。广告机如果支持上传 H5 文件夹，把整个 `H5小游戏_宣肺端午祝福` 文件夹上传。

## 页面定位

- 类型：领导展示型产品宣传互动页
- 屏幕：竖屏 2160 x 3840
- 主题：宣肺止嗽合剂 + 端午安康 + 轻知识科普
- 互动：触摸收集祝福签，45 秒结束生成祝福卡

## 素材说明

`assets` 中的宣肺产品图片来自本地已整理的投屏素材库。原始素材库未被修改。

端午互动装饰包括：

- `mugwort.svg`：艾叶
- `zongzi.svg`：粽子
- `sachet.svg`：香囊
- `blessing-card.svg`：祝福签

这些装饰用于 H5 首页陈列和游戏掉落物，避免重新生成产品包装文字，保证产品露出仍使用真实素材。
